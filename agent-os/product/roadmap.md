# Product Roadmap

1. [X] **Stock Data Visualization & Timeline** — Display interactive historical stock price charts with date range selection and ticker input. Users can view price movements over customizable time periods. `M`

2. [X] **Linear Regression & Trend Analysis** — Apply machine learning regression models (LightGBM) to historical stock data to identify trends and generate regression lines for technical analysis. `L`

3. [X] **Forecasting & Future Price Predictions** — Generate future price predictions using trained regression models, allowing users to visualize potential price movements for configurable forecast periods. `L`

4. [X] **Financial Statement Analysis** — Fetch and display key financial ratios (P/E, P/B, P/S, Debt Ratio, Liquidity Ratio) for any stock ticker, providing quick fundamental analysis insights with explanatory tooltips. `M`

5. [X] **Stock Screener with Custom Filters** — Allow users to screen stocks from predefined groups (S&P 500, Dow Jones) or custom ticker lists using configurable financial metric filters (P/E, P/B, ROE, etc.) to identify investment opportunities. `L`

6. [ ] **Portfolio Optimization Engine** — Optimize portfolio allocation using Modern Portfolio Theory and the Efficient Frontier, calculating optimal weights based on user-defined risk tolerance, target returns, and risk-free rate assumptions. `XL`

7. [X] **Hedge Analysis & Pairs Trading** — Analyze statistical relationships between two stocks using correlation and regression analysis to identify potential hedging opportunities and pairs trading strategies. `M`

8. [X] **Multi-language Internationalization** — Support multiple languages (English, Korean) with comprehensive translation coverage across all UI components and features for global accessibility. `S`

9. [ ] **User Authentication & Saved Portfolios** — Implement user account system allowing users to create accounts, log in, and save their custom portfolios, watchlists, and screening criteria for future sessions. `L`

10. [ ] **Real-time Data Integration** — Integrate real-time or near-real-time stock price updates through WebSocket connections or polling mechanisms to provide live market data instead of historical-only data. `XL`

11. [ ] **Alert & Notification System** — Build automated alert system that monitors stock prices, portfolio performance, and financial metric thresholds, sending notifications via email or in-app when user-defined conditions are met. `M`

12. [ ] **Advanced Technical Indicators** — Add support for custom and standard technical indicators (RSI, MACD, Bollinger Bands, Moving Averages) that users can apply and configure on stock charts for enhanced technical analysis. `L`

> Notes
> - Items 1-8 are already implemented and functional
> - Core MVP features (visualization, forecasting, portfolio optimization, financial analysis) are complete
> - Remaining items focus on enhancing user experience through personalization, real-time capabilities, and advanced analysis tools
> - Authentication and saved data (item 9) is the logical next step to enable persistent user experiences
> - Real-time data integration (item 10) would significantly enhance the tool's utility for active traders
